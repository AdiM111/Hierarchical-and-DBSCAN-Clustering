# Hierarchical-and-DBSCAN-Clustering
Implementation of Hierarchical Clustering (Agglomerative) and DBSCAN using Scikit-learn with Iris and make_moons datasets. Includes Dendrogram, PCA, Silhouette Analysis, and Cluster Visualization.

# Hierarchical Clustering and DBSCAN

This repository demonstrates the implementation of two popular Unsupervised Machine Learning clustering algorithms using Python and Scikit-learn.

## Algorithms Covered

- Hierarchical Clustering (Agglomerative)
- DBSCAN (Density-Based Spatial Clustering of Applications with Noise)

### Hierarchical Clustering
